# num_converter

a simple converter app for decimal-binary-octal-hexadecimal
##

#### Deploy in your VPS
````bash
git clone https://github.com/sarangsahajan/num_converter.git
cd num_converter
pip3 install -r requirements.txt
python main.py
````
#### Deploy on docker
````bash
git clone https://github.com/sarangsahajan/num_converter.git
cd num_converter
sudo docker compose up -d
````
